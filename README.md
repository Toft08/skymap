# Sky Map

A Flutter mobile app that shows a real-time, interactive map of the night sky. Point your phone at the sky and see the stars, planets, and constellations overlaid on a live view driven by your device's GPS, compass, and accelerometer.

## What's in this repo

| Path       | Description                                 |
| ---------- | ------------------------------------------- |
| `sky_map/` | Flutter application (source, assets, tests) |
| `task.md`  | Original project specification              |

## Requirements

- Flutter ≥ 3.x / Dart ≥ 3.11
- Android or iOS device with GPS, compass, and accelerometer

## Quick start

```bash
cd sky_map
flutter pub get
flutter run
```
